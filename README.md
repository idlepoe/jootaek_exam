주택관리사보수능력검정 **1차** 기출문제를 회독·모의고사 형태로 학습할 수 있도록 만든 **광고 없음** 학습 앱(웹·모바일) 프로젝트입니다.

## 앱 소개

- 회차별 기출 **회독**: 과목·시험 회차를 고른 뒤, 지문과 보기를 보며 **정답을 바로 확인**하며 반복 학습합니다.
- **모의고사**: 문항을 풀고 채점·이력 확인이 가능합니다.
- 회독 진행 위치 등 일부 학습 상태가 기기에 저장되어 이어서 학습할 수 있습니다.
- 광고가 포함되어 있지 않습니다.
- 모바일에서 보기 쉽도록 화면 폭·하단 조작 등을 구성했습니다.

## 이런 분께 추천합니다

- 주택관리사보 1차 기출을 **빠르게 여러 번** 훑고 싶은 분
- 문제·보기·정답을 **세트로 외우는** 방식이 잘 맞는 분
- 광고 없이 **집중**해서 보고 싶은 분

## 학습 흐름(요약)

1. 메인에서 과목을 고릅니다.
2. **회독**: 시험 회차를 선택해 문항을 넘기며 정답 표시를 확인합니다. **모의고사**는 별도 흐름으로 진행합니다.
3. 필요하면 설정에서 글자 크기·하단 버튼 순서 등을 조절합니다.
